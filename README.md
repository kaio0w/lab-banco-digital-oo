#  Lab Banco Digital OO

Projeto de estudo de Programação Orientada a Objetos em Java, simulando um sistema de banco digital.  

---

##  Sobre o projeto

O sistema simula funcionalidades básicas de um banco, como:

- Criação de contas
- Operações financeiras (depósito, saque, transferência)
- Regras de negócio e validações
- Uso de exceções personalizadas
- Estrutura baseada em orientação a objetos
